# Sistema de Sorteio – GTI 4º Período

Projeto desenvolvido no 4º período do curso de Gestão da Tecnologia da Informação, sob orientação do professor **Marcos Wesley**. A aplicação utiliza **Node.js**, **Express**, **Express-Handlebars** e **MySQL** para criação de um sistema de sorteio funcional.

## Funcionalidades

* Cadastro de participantes
* Listagem dos participantes
* Sorteio aleatório
* Páginas dinâmicas com Handlebars
* Integração com banco de dados MySQL

## Tecnologias Utilizadas

* Node.js
* Express.js
* Handlebars
* MySQL
* Nodemon

## Como Executar

```bash
npm install
npm run serve
```

Acesse no navegador: `http://localhost:5000`

## Aprendizados

* Rotas com Express
* Templates com Handlebars
* CRUD básico com MySQL
* Organização de projetos Node.js
* Boas práticas com Git/GitHub
